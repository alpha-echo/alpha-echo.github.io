LogToExcel

LogToExcel est un programme permettant de convertir des fichiers catalina.out en graphique Excel. Durant ce développement, un système de Kanban a été utilisé afin de planifier les tâches, et les avancées ont été régulièrement présentées devant mon tuteur au cours de mon stage pour aider à faire évoluer le programme.

Il y a eu en tout quatre demandes de version du programme, une par semaine de stage, la dernière n'ayant pas pu être développée.

Première version

Dans sa première version LogToExcel ne prenait en compte qu'une seule journée de log pour en faire une courbe. De plus durant cette semaine de stage j'ai pu en apprendre plus sur Tomcat qui est un serveur HTTP avec un conteneur de servelets (une classe java pour créer des données dynamiques).

Le programme est composé de 3 fichiers Java (Main, LogEntry et Excel) ainsi que d'un README.

  • Main.java : Fichier principal qui permet de lancer le programme. Il crée tout d'abord une fenêtre pour choisir le fichier catalina.out à analyser, puis crée une liste de tableaux de type LogEntry et les remplit avec les informations nécessaires (les différents niveaux des messages, INFO, WARNING et ERROR). Ensuite, il trie la liste de tableaux en fonction du niveau de sévérité du message. Enfin, il extrait de la liste le message de chaque ligne afin de l'envoyer dans Excel.java, qui permet de créer le fichier Excel.
  • LogEntry.java : Fichier de classe Java qui permet de stocker dans une liste le message, le type d'anomalie (Info, Warning et Error), l'hôte d'où provient le message ainsi que la date du message.
  • Excel.java/ : Fichier qui permet la création du fichier Excel grâce à la librairie ApachePOI. Il crée les différentes colonnes avec leur titre puis récupère les informations dans LogEntry pour les inclure dans le fichier. Ensuite, il compte le nombre d'occurrences de chaque type de criticité. Enfin, il fait apparaître une fenêtre pour permettre l'enregistrement du fichier Excel.
  • README-V1.nfo : Résume brièvement ce que fait le programme
Les fichiers

Les fichiers sont disponibles sous la forme d'un fichier .ZIP contenant un également exemple de fichiers Excel généré par le pogramme.

Deuxième version

La deuxième version est séparée en deux version les deux permettant d'envoyer le fichier Excel par mail mais la 2.2 permet l'envoi à plusieurs destinataires

Le programme se compose désormais de 4 fichiers avec l'ajout de SendMail.java

LogToExcel V2.1

  • Excel.java : Permet désormais de demandé via une fenêtre les identifaint de l'adresse mail de l'utilisateur pour les envoyer à SendMail.java
  • SendMail.java : Classe java qui permet grâce à javax.mail d'enbvoyer le fichier Excel en pièce jointe à un destinataire avec comme corps du message le résumé du graphique.
Les fichiers

Les fichiers sont disponibles sous la forme d'un fichier .ZIP contenant un également exemple de fichiers Excel généré par le pogramme.

LogToExcel V2.2

  • Excel.java : Permet désormais de demandé via une fenêtre les différents destinataires à qui envoyer le mail via SendMail.java
Les fichiers

Les fichiers sont disponibles sous la forme d'un fichier .ZIP contenant un également exemple de fichiers Excel généré par le pogramme.

Troisème version

Dans sa troisième version LogToExcel permet désormais d'analyser plusieur jour contenu dans un seul log catalina.out et de crée un graphique par jour en plus d'un graphique genéral pour l'ensemble du log

Le programme est toujours composé de 4 fichiers java ainsi que d'un README.

  • Main.java : Le fichier oermet désormais de trier un liste de liste avec comme critère les jour. Et c'est désormais lui qui demande les identifaints et le nombre de destinataires si le fichier doit être envoyé par mail.
  • Excel.java : Le fichier crée désormais une première page contenant un graophique avec les occurances de toutes les criticités du log sur l'ensemble des jours, ainsi qu'une page avec un graphique camembert par jour.
Les fichiers

Les fichiers sont disponibles sous la forme d'un fichier .ZIP contenant également exemple de fichiers Excel généré par le pogramme.

Quatrième version

Dans sa quatrième version LogToExcel était censé permettre de transformé le fichier Excel en fichier HTML mais cela n'a pas pu être réalisé par manque de moyens.
