Mediatek Documents

Cette application exploitant une API REST à été réalisée dans le cadre d'un atelier profesionnel

MediaTek86 constitue un réseau responsable de la gestion des médiathèques dans le département de la Vienne. Sa principale mission est de coordonner les emprunts de livres, de DVD et de CD entre les différentes médiathèques tout en promouvant le développement de ressources médiathèques numériques pour l'ensemble du département. MediatekDocuments est une application permettant de gérer le catalogue d'une médiathèque. Il s'agit d'un client lourd développé en C# qui communique avec une API hébergée sur Internet.
